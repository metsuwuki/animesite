# animesite.github.io

Современный аниме-лендинг с призматическим интерфейсом, кастомным шрифтом и плавными анимациями.

## Особенности
- Мягкий призматический дизайн, радужные переливы
- Кастомный display-шрифт (MADE Carving Soft Outline)
- Локальный fallback-шрифт для кириллицы (Manrope)
- Анимированные кнопки и карточки с shimmer-эффектом
- Видео-блок 720x1280 с адаптивным отображением
- Оптимизация под производительность (prefers-reduced-motion, минимизация blur)
- Полностью локальные ассеты, без внешних зависимостей

## Структура проекта
- `index.html` — основной HTML-файл
- `assets/css/style.css` — базовые стили
- `brand-font.css` — кастомные шрифты и override-стили
- `assets/video/` — видео для блока
- `public/fonts/` — локальные ttf-шрифты
- `unpkg.com/`, `fonts.googleapis.com/`, `fonts.gstatic.com/` — локальные копии внешних ресурсов для оффлайн-работы

## Запуск и сборка
1. Откройте проект в редакторе или разверните на GitHub Pages.
2. Для локального запуска используйте любой http-сервер (например, Live Server или Vite):
	- `npx vite preview` или `npx serve .`
3. Для продакшн-сборки (если используется Vite):
	- `npm run build`

## Кастомизация
- Для замены бренд-логотипа — замените изображение в секции header.
- Для смены видео — положите новый файл в `assets/video/` и обновите путь в разметке.
- Для изменения шрифта — замените ttf-файл в `public/fonts/` и обновите `brand-font.css`.

## Лицензия
Проект распространяется свободно для некоммерческого использования. Все сторонние шрифты и ассеты — согласно их лицензиям.
